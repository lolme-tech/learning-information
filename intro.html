自己紹介

はじめまして。
現在大学4年生で大学ではプログラミングを学んでいます。
大学2年の時に、このままただの大学生活を送るのは違う気がすると思い、軽い気持ちで資格の勉強を始めました。
当時は資格の知識もなかったためにインターネットで色々調べ、IPAが主催しているIT資格、ITパスポートに出会い勉強を始めました。
それがきっかけで勉強に火が付き、今に至るまでほぼ毎日かかさず勉強をする日々を送っています。
下記に取得した資格や経緯などを記します。

取得資格と経緯

2019年8月 ITパスポート取得

初めての資格試験ということでどのように勉強をしたらよいのか、悩んでいました。
参考書一冊を4週ほどやりこみ、受かる見込みで受験をした結果、不合格でショックだったのを覚えています。
そこから1~2か月ほど別の参考書と過去問をやりこみなんとか合格しました。

2019年11月 基本情報技術者試験取得

当時は毎日夜遅くまでアルバイトをしていたので、勉強をする時間を確保するのが大変でした。
アルバイトから帰宅して1時間ほど勉強、通学の電車内で勉強など時間を貴重に扱っていたのを鮮明に覚えています。
午前の問題はITパスポートと内容がほぼ同じなので、過去問をひたすら解き、午後に関してはネットで得点を取りやすい選択問題などを
調べて分析しつつ演習をしていました。
ちなみに選択問題は表計算を選択しました。当時はまだPythonがなかったですし、大学で学んでいる
プログラミング問題は難易度が高いと聞いたので簡単な表計算にしました。

2020年10月 TOEIC650点取得

この年は、コロナが流行し応用情報技術者試験の春の部が中止になったために自粛中暇すぎて
応用情報の勉強と並行して5月ごろから毎日30分英語の勉強とOracle JAVA Silverの勉強をするようになりました。
途中経過としてTOEICを受けてみようということで5か月30分毎日勉強した結果650点でした。

2021年1月 Oracle JAVA Silver取得

コロナの自粛で暇だったため、なにか資格を取ろうと思い、始めたのがこの資格です。
資格内容としてはJAVAの勉強というよりはオブジェクト指向やクラス図の使い方を学べたので、
それらのスキルを客観的に示す物差しとして取得の意義を感じる資格だと思います。

2021年3月 TOEIC710点取得

毎日30分英語勉強の経過観察のために受験しました。およそ10か月間毎日30分勉強したことになります。

2021年4月 TOEIC765点取得

毎日30分英語勉強の経過観察のための受験ですが、前回の受験から1か月しか経っていないため、あまりスコアも
伸びないだろうと思っていたのですが、55点も伸びました。教習所で空いた暇な時間を英語の勉強に当てていたのが成果につながったのかもしれません。

2021年5月 普通自動車第一種運転免許(AT限定)取得

この資格に関しては、特にいうこともありませんが、社会人になると取りづらくなるので
自粛期間の暇を有効に使うため取得しました。

2021年12月　応用情報技術者試験取得

4回目の受験にてようやく取得しました。一度目の受験は新型コロナの影響で中止になってしまいました。
その後、2回目は午後が37点で不合格、3回目は午後が47点で不合格でした。
4回目にて気持ちを入れ替えて、1から勉強し直し、 念願の合格を果たしました。これにはさすがにガッツポーズ。
応用情報を取得するうえで、2年間勉強したノウハウを別のリンクで紹介していきたいと 思います。
応用情報取得までの流れ⇒応用情報技術者試験合格体験記

2022年6月　TOEIC830点取得

約1年ぶりにTOEICを受験しましたが、毎日の継続が結ばれました。
ほんとに800点以上を毎日30分勉強するだけで取得できることに、継続力の大切さを実感することが出来ました。
英語力が全くなかった私が毎日30分だけ何かを続けることで英語力が身についたという経験は
今後も、何かを身に着けたいと思う私を勇気づけると思います。
大学院を卒業するまでは引き続き毎日30分英語を勉強し続けようと思います。
